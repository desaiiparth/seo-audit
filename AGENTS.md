# SEO Audit Agent Instructions

This project is a comprehensive SEO audit agent.

The agent must:
- Crawl website URLs from a sitemap.
- Extract SEO data from each page.
- Generate CSV and Markdown reports.
- Avoid generic SEO advice.
- Tie every recommendation to real evidence from crawl data, Google Search Console, PageSpeed, GA4, or page content.
- If evidence is missing, label the diagnosis as “unconfirmed.”
- Include indexed/not indexed pages, metadata, H1/H2 analysis, keyword mapping, performance diagnosis, CRO suggestions, new SEO opportunities, AEO/GEO recommendations, and a 30/60/90-day roadmap.
